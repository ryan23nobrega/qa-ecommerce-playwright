# Estratégia de Testes

## Objetivo

Garantir a qualidade das funcionalidades críticas do sistema Automation Exercise através de testes funcionais, exploratórios, API e automação.

---

## Escopo

### Em Escopo

- Cadastro de usuário
- Login
- Busca de produtos
- Carrinho de compras
- Checkout
- Logout

### Fora de Escopo

- Performance
- Testes Mobile
- Testes de Segurança Profundos

---

## Tipos de Testes

### Funcionais

Validação dos requisitos de negócio.

### Exploratórios

Identificação de comportamentos inesperados.

### Regressão

Garantir estabilidade após mudanças.

### Smoke Test

Validar rapidamente funcionalidades críticas.

### API

Validação dos endpoints disponíveis.

### Automação

Execução automatizada dos fluxos críticos.

---

## Critérios de Entrada

- Ambiente disponível
- Requisitos definidos

---

## Critérios de Saída

- Todos os testes executados
- Bugs críticos corrigidos
- Evidências anexadas
